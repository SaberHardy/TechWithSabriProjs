1. HTML structure and Tags

Basic Structure

Every Html Document Starts with the following structure:

            <!DOCTYPE html>
            <html>
                <head>
                    <title>My webpage</title>
                </head>
                <body>
                    Body content starts here
                </body>
            </html>

Common Tags

  • <p>: Paragraph
  • <h1> to <h6>: Headings
  • <div>: Division of content
  • <span>: Inline container
